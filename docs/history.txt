== История

Данные продукт используется в качестве автосборки проектов в РБК с 2012 года. 

Изначально для сборки проектов использовался ANT, но так по некоторым причинам был произведен переход на Phing.
.Отрицательные стороны Ant
* медленная скорость работы
* необходимость устанавливать дополнительные пакеты *jar*
* сложность расширения
* трудности при работе с шаблонами
