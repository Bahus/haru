== Описание

Универсальный автосборщик. Набор инструкций для link:http://www.phing.info[Phing], предназначенный
для автоматизации сборки и установки проектов.

== Требования к ПО
PHP >= 5.3

== Установка
Скачайте и распакуйте архив необходимой версии программы link:https://github.com/TheRatG/haru[Haru]
  
Или
  
Скопируйте репозиторий
[source,bash]
----
git clone git://github.com/TheRatG/haru.git
----

== Быстрый старт

Выполните следующие команды:
[source,bash]
----
git clone git://github.com/TheRatG/haru.git
cd haru
./phing born -Dpath=/www/kin -Dhost=kin.dv -Dlib=kin -Dget-build.deploy.setup=1
----
    
*Учебный проект готов*
  
Для запуска сайта настройте виртуальный хост: 
[source,bash]
----
## создайте ссылку на кофигурационный файл для apache
## например
sudo ln -s /www/kin/etc/kin.conf /usr/local/apache2/conf.d/kin.conf
----

== Проблемы

Зависает скрипт выката. 

Такая ситуация возникает в случае, когда необходимо интерактивое действие пользователя. 
Например запрос логина или пароля при клонировании репозитория. Попробуйте выполнить последнюю команды из 
отладочного вывода процесса.


== История

Данные продукт используется в качестве автосборки проектов в РБК. 

Изначально для сборки проектов использовался ANT, но так по некоторым причинам был произведен переход на Phing.
.Отрицательные стороны Ant
* медленная скорость работы
* необходимость устанавливать дополнительные пакеты *jar*
* сложность расширения
* трудности при работе с шаблонами
